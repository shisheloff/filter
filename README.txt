---filter.py 
Парсинг полей
Приход на 514 или 3515 порт
Запись в эластик

---main.cfg
Настройки записи в эластик и его template + название template
port эластика
ip Эластика

---corel.py
Обработка событий и нахождение инцидентов
Получение событий эластика и обработка одиночных событий путем сохранения номера индекса для последующей проверки

---corel.cfg
Конфиг файл для определения папки с правилами типа: ./rules/1.def (mes:error auth)
**Колво буфера событий для обработки каскады событий разом.

---Эластик серч json
if - номер события
time - время события
src.ip - источник события
mess - сообщение события
corel - Наименование трегера правила (or none)


Структура:
/bin/
--main.py
--syslog.py (listen 514)
--winlog.py (listen 3515)
--corel.py (listen 9150)
/settings/
--main.cfg
--corel.cfg

Используемые порты:
input 514 - syslog.py - out 9200
input 3515 - winlog.py - out 9200
input 9200 - elasticsearch
use 9200 - corel.py
